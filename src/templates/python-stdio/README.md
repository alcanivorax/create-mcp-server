# **PROJECT_NAME**

A minimal MCP server implemented in Python using stdio transport.

## Requirements

- Python 3.9+

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Running the server

```bash
python src/main.py
```

## Claude Desktop configuration

```bash
{
  "command": "python",
  "args": ["src/main.py"]
}
```
